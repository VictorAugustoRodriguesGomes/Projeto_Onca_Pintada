Conversor de Moeda Internacional.

Bem-vindo à ferramenta de conversão de moedas internacionais!

Valor a ser convertido:

R$:

de:

R$ - Real
$ - Dólar
€ - Euro

para:

R$ - Real
$ - Dólar
€ - Euro

Usamos a API: docs.awesomeapi.com.br

Sobre o Projeto

Olá! Meu nome é Victor Augusto e sou responsável por desenvolver esse projeto nomeado com Projeto Onça-Pintada(utilizei esse nome genérico para melhorar organização pessoal).

O objetivo é demonstrar as minhas habilidades como desenvolvedor.

Desenvolvi esse projeto que converte valores entre moedas.

Para isso, basta informar o valor que pretende converter e selecionar a moeda de origem e a moeda que pretende ser convertida.

As moedas utilizadas são: Dólar, Euros e Reais, no futuro pretendo adicionar mais moedas.

O projeto foi desenvolvido utilizando a API Awesome com HTML, CSS e JS (JavaScript).
